# Dimension Fork
 
## Asynchronous chat meets tabletop style roleplay mechanics.

Your friends all use Discord and Slack, but they're too dang busy to all get together around a table for some good old fashioned RP?

Let's get our chocolate in our peanut butter and try roleplay like it's 202X! Schedule a session and you can jam like you used to, but without the need to co-locate. Or, you can try a slow-burn style and keep the campaign running in the background, pop in and take actions in character when you feel like it.

Roleplay that fits in your busy life... *and there was much rejoicing!*
 
 
 
Setup
---
 
```
npm install
```

 
Compile
---
 
```
npm run compile
```
